# 基于uniapp实现聊天程序
## 目前实现功能
- 登录注册
- 添加好友/创建群
- 加入群
- 单聊
- 群聊
- 退出
- 上传头像
- 消息震动
- 发送表情
- 图片消息
- 验证码
- 语音消息
- 消息持久化
- 列表消息气泡
